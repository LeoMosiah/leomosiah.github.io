Hick's Law

Microsoft Github

Github is a source code storage and hosting platform with version control that makes use of git. It allows its users to have their source code hosted on private or open-source projects. It is currently one of the largest platforms for hosting source code.

As each user can have an unlimited number of source code repositories they make use of the Hicks law to make the information more user friendly. On the home page it shows only your recently used repositories and just below information about your activity and the people you dry. In addition to navigating to a more complete list of repositories or account settings, and organizations that each user has..

Visual Hierarchy

Nintendo Nintendo E-Shop

Nintendo is a developer and publisher of video games and consoles based in Kyoto, Japan. As it is an international company it has a website where all its products are available for sale. In their E-Shop they make a great use of visual hierarchy to draw attention to their main products. In the screenshot it was clear the emphasis that is given for the sale of the nintendo switch console that was recently made available in Brazil.

Fitt's Law

Udemy Udemy Courses

Udemy is a distance learning e-learning platform, where people can learn and teach through it various types of different knowledge. For having several courses to be shown, udemy used the fitts law very well to have a website where the most important interactions are within the user's reach. Soon on the home screen the research input is very close to the end of hers, having access easy for quick search, and also all site navigation can be found in the menus that are on top of the site, also easily reached with your fingers.
